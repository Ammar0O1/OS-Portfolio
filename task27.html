What is sleep()?

The sleep() system call is used to pause the execution of a program for a specific number of seconds. When a process calls sleep(), it stops running for that time and lets the operating system give the CPU to other processes.

In simple words, sleep() tells the program: “wait here for a while, then continue.”

Why is sleep() useful?

  • It delays a program for a fixed amount of time.
  • It helps simulate waiting in programs.
  • It prevents a program from using the CPU continuously when it does not need to.
  • It is useful in examples about processes, timing, and scheduling.

How sleep() works

When sleep(5) is called, the running process is moved into a waiting or sleeping state for 5 seconds. During this time, the CPU can work on another process. After the time is finished, the process becomes ready again and continues from the next line of code.

C Example

#include <stdio.h>
#include <unistd.h>

int main() {
    printf("Program started\n");

    sleep(3);

    printf("Program continued after 3 seconds\n");

    return 0;
}

In this example, the first message prints immediately. Then the program waits for 3 seconds. After that, the second message is printed.

Important Note

The sleep() function does not delete or stop the program completely. It only pauses the process for the selected time. When the time finishes, the program continues normally.
